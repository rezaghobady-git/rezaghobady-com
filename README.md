# rezaghobady.com

Personal website for Reza Ghobady — entrepreneur, founder of [KaavOps](https://kaavops.com) and [Simine Paris](https://simineparis.com).

## Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v4 + CSS variables
- **Language:** TypeScript (strict)
- **Content:** MDX (blog posts)
- **Deployment:** Hostinger VPS via GitHub Actions

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run start
```
