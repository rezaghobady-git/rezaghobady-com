# Design System Document

## Theme Overview
This document outlines the core visual theme of our design system.

## Color Palette
The color mode for the UI is **light**.
- **Primary Color**: `#E76F51` - A vibrant and engaging hue for primary actions and branding.
- **Secondary Color**: `#8D8172` - A versatile supporting color for secondary elements and complementary details.
- **Tertiary Color**: `#E9C46A` - An accent color used for highlights, alerts, or decorative elements.
- **Neutral Color**: `#FDFBF7` - A light, clean base color for backgrounds and general UI surfaces.

## Typography
- **Headlines**: `plusJakartaSans` - Used for all prominent titles and headings, conveying a modern and clear tone.
- **Body Text**: `inter` - Applied to paragraphs and general content for excellent readability across all sizes.
- **Labels**: `inter` - Utilized for form labels, buttons, and other interactive text elements.

## Shape and Form
The UI elements feature a **moderate** level of roundedness (value: 2), offering a balanced and approachable aesthetic without being overly soft or sharp.

## Spacing
The design system employs a **normal** level of spacing (value: 2) between and within UI elements, ensuring clarity and comfortable visual separation. This provides a balanced layout that is neither too dense nor excessively spacious.
